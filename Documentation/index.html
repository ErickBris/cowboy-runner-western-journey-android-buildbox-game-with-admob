How to import project in Eclipse? - top

  1. Open Eclipse, and from menu select File > Import.
  2. Under Android select "Existing Android Code Into Workspace" then click Next.
  3. Browse to the Root Directory of the project (make sure you select Android folder).
  4. Select both projects "com.getemplate.cowboyrunner" and "google-play-dervices_lib" then click Finish
  5. You may need to clean the project first, to do that, from the menu: Project > Clean... > Clean All Projects > OK (wait until its finished).
  6. Now if you still get error library path broken so you have to first remove the library then you have to add it again, to that you have to Right Click Project > Properties > Android > at bottom of page you see to library if you have red mark then select it and remove then again add library

How to change the package name? - top

You can change it easily from Eclipse. Here are the steps:

  1. Right Click 'com.getemplate.cowboyrunner' project from Package Explorer > Android Tools > Rename Application Package > Rename package name > OK.
  2. Refactoring window will apear, Click Finish > Yes.
  3. Rename the project: Right click on project > Refactor > Rename...
  4. Enter new name > OK
  5. Do the same with the package "src/com.getemplate.cowboyrunner"

Watch this video for more details.



How to change App Name and Google Play Id? - top

  1. In Eclipse open: com.getemplate.cowboyrunner > res > values > strings.xml
  2. Change "Cowboy Runner" to the name you choosed for your game.

    <?xml version="1.0" encoding="utf-8"?> <resources>
    <string name="app_name">Cowboy Runner</string>
    <string name="app_id"></string>
    </resources>

How To Change Application Icon?

How To Change Application Splash Screen?

You can find Splash Screen in: "Cowboy Runner (Eclipse Project)\android\assets"

How To Change Application Graphics?

You will find all image into "Cowboy Runner (Eclipse Project)\android\assets\data\atlases" Just create your own graphics and replace them in atlases.

How To Change Application Sounds

You will find all Sounds into "Cowboy Runner (Eclipse Project)\android\assets\data\sounds" Just replace with your own sounds.
